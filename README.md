# Critical-Actin-in-Dendritic-Spines
2D simulations of actin dynamics in dendritic spines
Code used in Bonilla-Quintana, M., Wörgötter, F., D'este, E., Tetzlaff, C. and Fauth, M., 2020. Actin in Dendritic Spines Self-Organizes into a Critical State. BioRxiv. doi: https://doi.org/10.1101/2020.04.22.054577 
Run Main.m for simulating dendritic spine shape fluctuations or Main_withoutFeedback.m for a model without feedback in the branching rate of the barbed ends.
For plotting use plot_simulation.m
checkCritical.m tests whether actin self-organizes in a critical state. 
To produce blurry images of the simulated dendritic spine run plot_blurry.m
